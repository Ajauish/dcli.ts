# About
<b>Dcli.ts is a all-in-one project generator for developers, made by developers.<b>
- Easily generate
- Simple Interface
- Supports latest versions of technologies

# Installiation 
Use latest version of node. Package MUST be installed global.
```
npm i dcli.ts -g
```
